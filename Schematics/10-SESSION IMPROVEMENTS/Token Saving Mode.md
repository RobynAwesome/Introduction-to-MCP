# Token Saving Mode

## Rule

- token-saving mode is mandatory outside Plan Mode and outside Lead-only sessions with Master

## What It Means

- Lead delegates heavy-token work where possible
- blocked lanes go to standby instead of being left half-active
- long proofs and research packs belong to explicit passes, not casual drift

## Exemptions

- Plan Mode
- Lead-only sessions with Master

## Failure Signal

- premium context is being spent on work that could have been delegated or paused
