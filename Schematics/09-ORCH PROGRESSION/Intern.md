# Intern

## Purpose

- perform bounded documentation or audit tasks inside explicit scope

## Allowed Responsibilities

- update a note
- draft a checklist
- label a blocker clearly

## Blocked Responsibilities

- no repo-wide decisions
- no autonomous architecture changes

## Supervision Model

- task-by-task review by Lead

## Promotion Criteria

- completes bounded note tasks cleanly
- uses evidence and dated logging correctly

## Demotion Trigger

- phantom completion
- file drift outside assigned scope
