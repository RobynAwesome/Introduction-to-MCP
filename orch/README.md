# orch

Multi-AI Collaboration Platform (MCP) – a central orchestrator that runs multiple AIs in a private group and learns from their discussions.

## Commands
- `orch agents config`: Configure an AI agent.
- `orch serve launch`: Start a new simulation.
- `orch chat log`: View discussion logs.
- `orch learn datasets`: Access the knowledge base.

## Tech Stack
- CLI: Typer & Rich
- Models: LiteLLM
- Configuration: Pydantic-Settings
